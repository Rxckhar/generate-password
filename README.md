# generate-password

Получить список всех действующих библиотек для проекта:
```bash
pip list
```

Учесть все изменения:
```bash
git add .
```

Коммит:
```bash
git commint -a -m "New version 0.0.0"
```

Отправить изменения:
```bash
git push
```

 Указать версию сбрки:
 ```bash
 git tag v0.0.0
 ```

Отправляем новую версию:
```bash
git push origin --tags
```
